# Credit Risk Scoring — Home Credit Default Risk

## Overview
Machine learning pipeline to predict loan default probability using Home Credit dataset.

## Results
| Model | ROC-AUC |
|---|---|
| Logistic Regression | 0.7502 |
| Random Forest | 0.7336 |
| XGBoost | 0.7584 |
| **LightGBM** | **0.7685** |

## Stack
Python, Scikit-learn, XGBoost, LightGBM, SHAP

## Dataset
[Home Credit Default Risk - Kaggle](https://www.kaggle.com/competitions/home-credit-default-risk)
